## ðŸŒŸ Overview
In an era of increasing document forgery, this project provides a decentralized solution for issuing and verifying academic credentials. By leveraging blockchain technology, we transform static paper certificates into **tamper-proof digital assets** that can be verified instantly by employers and institutions worldwide without manual intermediaries.

> **The Goal:** To eliminate certificate fraud and reduce verification time from weeks to seconds.
